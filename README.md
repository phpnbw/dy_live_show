# 油猴插件：dy_live_show
### 抖音直播间人数超过10万+ 就不显示实际人数了（只显示：10万+）
### 这个插件可以让直播间显示详细的人数，5秒更新一次数据。

## 功能特点
- 实时显示直播间真实观看人数
- 每5秒自动更新数据
- 点击人数可查看详细统计，可查看直播间人数变化趋势，粉丝数，直播点赞数等数据
- [👉详细数据地址，点击跳转👈](https://douyin.phpnbw.com)
